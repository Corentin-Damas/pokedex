#Build a Pokedex

Small fun project using az REPL, Read-Eval-Print Loop, that takes user input, evaluates it, and returns the result to the user.
The Pokedex-like REPL in Go that uses the PokeAPI to fetch data about Pokemon.

From the CLI the user may, search in different game locations, scan for pokemons, capture the pokemon and get some data from them.  

### Build with 

- Go
- CLI
- http networking
- data serialization
